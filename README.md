# 🚻 Public Toilet Finder

> 현재 위치를 기반으로 가장 가까운 공공화장실을 쉽고 빠르게 찾을 수 있는 위치 기반 서비스

## 📌 프로젝트 소개

외출 중 갑작스럽게 화장실이 필요한 상황은 누구나 한 번쯤 경험합니다.
하지만 주변에 공공화장실이 어디 있는지 찾기 어렵거나, 운영 여부 및 시설 정보를 확인하기 힘든 경우가 많습니다.

**Public Toilet Finder**는 이러한 불편함을 해결하기 위해 제작된 위치 기반 서비스입니다.

사용자는 현재 위치를 기준으로 가장 가까운 공공화장실을 검색하고,
길 안내를 받을 수 있으며, 화장실의 시설 정보와 운영시간까지 확인할 수 있습니다.

향후에는 사용자 리뷰와 AI 추천 기능을 통해 더욱 편리한 공공화장실 탐색 서비스를 제공하는 것을 목표로 합니다.

---

# 🎯 프로젝트 목표

- 공공화장실을 빠르게 검색
- 사용자 위치 기반 최단 거리 안내
- 운영시간 및 시설 정보 제공
- 사용자 리뷰를 통한 신뢰성 확보
- 공공데이터를 활용한 생활 편의 서비스 구현
